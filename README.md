## What is Audify?

Audify is a free YouTube Music client for Android — built from the ground up with a focus on design, speed, and the features that actually matter.

No subscriptions. No ads. No bloat. Just music.

The UI is crafted around a **liquid glass / glassmorphism** aesthetic — smooth, modern, and easy on the eyes. Every screen, every animation, every detail is designed to feel premium without costing a penny.

## Getting Started

**1. Download the APK**

Head to the [Releases](https://github.com/pihuwingj/Audify/releases/latest) page and download the latest `.apk` file.

**2. Enable Unknown Sources**

Go to `Settings → Security → Install unknown apps` and allow your browser or file manager.

**3. Install & Enjoy**

Open the downloaded APK, install it, and you're good to go.

```
Minimum requirement: Android 8.0 (API 26) or higher
```

## FAQ

**Is Audify free?**
Yes, completely free. No in-app purchases, no subscriptions, no ads — ever.

**Does it need a YouTube Music subscription?**
No. Audify works without a premium subscription, though some content may be limited by YouTube's own restrictions.

**Is my account safe?**
Audify uses YouTube Music's standard login flow. Your credentials are never stored on any external server.

**Will there be updates?**
Yes — updates are released regularly with new features, bug fixes, and improvements.
